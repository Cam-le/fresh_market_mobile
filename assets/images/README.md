# Assets

Place local image assets here if you want to bundle them with the app.

## Current usage
All product and banner images are loaded from the network (Unsplash).
To use local images instead, add them here and reference via:

```dart
Image.asset('assets/images/your_image.png')
```

## Logo
Replace `assets/images/logo.png` with your actual app logo (recommended: 512×512 PNG).
